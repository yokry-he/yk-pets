# NOVA Browser Agent v0.6.1 Release Notes / 发布说明

## 中文

v0.6.1 是动画连续性、主动发光和生活姿态修正版。

- 鼠标移入/移出不再打断主动画。
- 新增带优先级的单项动作队列和 260ms 中性稳定阶段。
- 普通闲时动作缩短到 10～18 秒，生活、高能和彩蛋动作也调整了间隔。
- 新增好奇扫描、爪尖点点、触角聚能、尾光流动。
- 触角根部下移并嵌入头部。
- 触角、尾巴和胸口核心支持主动发光联动。
- 云朵午睡改为明确侧躺姿态。

## English

v0.6.1 improves motion continuity, explicit glow control, and lifestyle posing.

- Pointer enter/leave no longer interrupts primary motions.
- Added a priority-aware single-item motion queue and a 260ms neutral settling stage.
- Normal idle cadence is now 10–18 seconds, with shorter lifestyle, high-energy, and easter-egg windows.
- Added Curious Scan, Paw Tap, Antenna Charge, and Tail Glow Flow.
- Antenna roots now embed into the head surface.
- Antennae, tail, and chest core support directly triggered glow choreography.
- Cloud Nap now uses a clear side-lying pose.
