# NOVA Browser Agent v0.4.3 验证报告

## 构建方式

本版本以 v0.4.2 源码为基线，删除旧的 `apps/extension/.output` 和 `apps/extension/.wxt` 后执行：

```bash
wxt prepare
vue-tsc --noEmit
wxt build -b chrome
wxt zip -b chrome
```

构建由 Vue/TresJS 源文件生成 Chrome MV3 生产包，没有向旧生产 JavaScript 叠加运行时补丁。

## v0.4.3 专项检查

- 招手包含抬起、腕部摆动和回落阶段；
- 招手前臂竖起，摆动集中在爪端；
- 伸懒腰双爪上举；
- 伸懒腰头部后仰并向上拉伸；
- 能量球放大并下移；
- 玩球期间头部和眼睛持续锁定球；
- 鼠标向下时头部和眼睛向下跟随；
- 视线加入中心死区和阻尼；
- 视线优先级不被鼠标覆盖；
- 吃饭增加抬高餐桌；
- 饭盆按嘴部锚点水平对齐；
- 吃饭时嘴部和食物表面距离缩短；
- 根包、扩展包和 Manifest 版本统一为 `0.4.3`。

## 自动视觉测试边界

容器内 Chromium 能启动，但访问本地视觉测试页面时受到环境管理策略阻止，返回 `ERR_BLOCKED_BY_ADMINISTRATOR`。因此本报告没有把自动截图列为视觉验收结果；实际安装后仍需对招手方向、伸展最大姿态、球的高度和餐桌间距进行人工观察。
