## 安全断言

- 本地 Host 不提供任意 Shell 或动态 Git 子命令。
- Git 使用参数数组和 `shell: false`。
- 提交使用 `--no-verify`，不执行仓库 Hook；验证必须显式上报。
- 推送仅允许显式远端名称和规范化 URL。
- 页面环境不接触 Git 工作区路径、HMAC 密钥或 GitHub 凭据。

## 未声称完成的内容
