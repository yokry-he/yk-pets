# NOVA Browser Agent v0.5.2 Release Notes / 发布说明

## 中文

v0.5.2 是文档与可维护性版本，不改变 v0.5.1 的网络规则和宠物行为。

新增：

- 详细中英文使用操作手册；
- 详细中英文网络实验室与 Mock 工作台操作手册；
- 详细中英文构建、打包和 Chrome Web Store 发布手册；
- 中英文故障排查、开发维护与代码注释规范；
- v0.5.0 网络实验室和 v0.5.1 Mock 工作台英文文档；
- 57 个手写 TypeScript/Vue/CSS 文件的双语职责头；
- 核心大文件的状态、生命周期、网络边界、动画阶段、模板和 CSS 分区注释；
- 文档完整性和源码职责注释自动检查。

## English

v0.5.2 is a documentation and maintainability release. Network-rule and pet runtime behavior remain the same as v0.5.1.

Added:

- Detailed Chinese and English user guides;
- Detailed Network Lab and Mock Workbench operations;
- Detailed build, packaging, and Chrome Web Store release guides;
- Bilingual troubleshooting, maintenance, and comment standards;
- English v0.5.0 Network Lab and v0.5.1 Mock Workbench documents;
- Bilingual responsibility headers for 57 handwritten TypeScript/Vue/CSS files;
- Section comments for state, lifecycle, network boundaries, animation phases, templates, and CSS in core files;
- Automated documentation-completeness and source-responsibility checks.
