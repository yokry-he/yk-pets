# v0.7.3 自适应渲染与 2D 降级

## 决策信号

- WebGL 不支持或 Context Lost：立即切换 2D；
- 用户手动选择：立即切换；
- 减少动态效果、Save-Data、低电量、低内存：在自动模式下建议 2D；
- FPS 或长任务：连续 3 次不健康才降级，连续 6 次健康才恢复；
- 默认 30 秒切换冷却，避免 2D/3D 来回抖动。

## 状态迁移

切换前读取当前渲染器 `snapshot()`，新渲染器 `restore()`。动作、情绪、说话状态、评分、视线和扩展字段均可保留。3D 创建或挂载失败时自动改用 2D。

## 2D 云狐

Canvas 2D 版本不依赖贴图和远程资源，使用路径、椭圆、贝塞尔曲线绘制身体、头、耳朵、尾巴、云朵和能量核心。它不是静态占位图，仍保留呼吸、招手、跳跃、旋转、思考、休息和发光反馈。
