# NOVA v0.6.8：审计子规则、尾尖与动作语音修复

## 交付范围

本版本在 v0.6.7 稳定基线上完成四项跟进修复：移除动作小气泡、增强深色背景下的尾尖可见性、修复内置动作语音播放链路，并把页面审计范围细化到每一条实际规则。

## 页面审计规则

- 共享领域层新增 16 条审计规则注册表，规则代码与所属分类只有一个共同真源。
- Side Panel 按性能、无障碍、SEO、最佳实践和 DOM 分组列出全部子规则。
- 父分类支持全选、全不选和部分选择状态；每个子规则可单独开关。
- 用户选择保存在 Chrome 本地存储中，并兼容迁移 v0.6.7 的分类级配置。
- Content Script 在执行规则前判断规则代码，未勾选的检查不会扫描、不会生成问题，也不会参与评分。
- 报告记录实际执行的规则，并明确区分“已检查”和“未检查”。

例如，取消“图片没有声明固有尺寸”后，`image-dimensions-missing` 不再执行，但图片懒加载、长任务和大型资源等其他性能规则仍可继续运行。

## 宠物反馈

- 动作期间的小型思考气泡及其状态、定时器、模板和样式已全部移除。
- 18 条预生成中文动作语音继续随扩展内置，不依赖在线 TTS 服务。
- 语音资源在覆盖层挂载后预取；用户点击动作时立即恢复 Web Audio 上下文，再解码缓存并播放对应 MP3。
- 新动作会停止上一段语音；加载或解码失败会写入可诊断日志，且不会回退到“滴”声。

## 尾尖效果

- 尾尖管体不再使用高透明材质。
- 尾尖中心使用不受灯光影响的不透明亮核，并关闭 tone mapping 以保持亮度。
- 外层光晕使用加法混合且不写入深度，普通状态持续可见。
- 尾光、尾巴旋风、能量爆发、星星杂耍和烟花动作继续驱动多色循环闪烁。

## 兼容性

本版本没有增加浏览器权限或运行时依赖。旧审计报告仍可显示；旧分类配置会在首次加载时安全映射为对应分类下的全部子规则。
