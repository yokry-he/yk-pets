# Pet Studio 高级交互人工验收

- 使用中文和英文关键词搜索鼻子、肚皮、颜色、尾巴、轨道和检查，确认进入正确编辑区；
- 在非输入状态按 `/` 聚焦搜索，按 `Escape` 清空搜索并退出对比；
- 开启经典对比，确认预览和控件进入只读经典状态，保存和编辑按钮不可误操作；
- 返回当前外观，确认所有编辑、草稿和撤销历史完整恢复；
- 在对比状态直接关闭页面，再打开 Studio，确认当前外观没有被经典配方覆盖；
- 使用鼠标和键盘激活头部、身体、尾巴和标志热点；
- 保存两个本地方案，分别应用和删除，刷新后确认剩余方案仍存在；
- 检查自动草稿时间、撤销/重做数量随操作更新；
- 在窄屏和减少动画模式下确认搜索结果、热点和方案操作可用；
- 确认没有新增网络上传、第二个 WebGL 场景、后台轮询或浏览器权限。
