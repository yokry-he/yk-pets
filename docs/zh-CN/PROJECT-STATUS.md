# NOVA 当前项目状态

## 当前基线

- 当前版本：`v0.6.10`；
- 主要交付：Chrome/Edge Manifest V3 浏览器扩展；
- 当前阶段：产品能力稳定、浏览器回归与缺陷修复；
- npm SDK、Web Component 和多框架适配暂不属于当前实施范围。

## 当前工作原则

- 修复必须覆盖真实运行链路，不能只调整界面表现；
- 保持现有规则和本地数据向后兼容；
- Domain 层不依赖 Vue、Chrome API、DOM 或具体 Agent Provider；
- 正式交付统一更新版本号、回归检查、验证报告和构建产物；
- 页面刷新、扩展重载和 Side Panel 重开后的状态必须可预测。

## 阶段完成标准

- 核心交互、动画和 Network Lab 缺陷完成浏览器安装回归；
- 手动新增、从请求生成、编辑、复制、启停和删除规则稳定可用；
- Fetch 与 XHR 的 Mock、延迟和完整 JSON 响应替换行为一致；
- 关键路径具备自动化回归；
- 全部 Workspace 类型检查、测试和生产构建通过；
- `v0.6.10` 可作为后续平台化拆分的稳定参照实现。

## 下一阶段方向

完成当前稳定化工作后，再进入 SDK 基础架构阶段：提取框架无关的宠物状态与动作运行时，定义稳定生命周期和能力接口，并逐步拆分渲染、Agent、主题、多语言与扩展能力。
