# Side Panel 任务工作台人工验收

## 导航与首页

- 打开 Side Panel，确认一级导航为首页、记忆、审计、网络、云灵；
- 首页显示当前页面标题和 URL；
- 创建当前页面记忆后，首页数量无需刷新扩展即可更新；
- 完成一次页面审计后，首页显示分数和问题数；
- 连接和断开 Local Agent，确认首页状态更新；
- 点击首页四个主动作，确认分别进入已有工作区或启动已有审计流程。

## 云灵工作区

- 音色、3D 开关、闲时动作、宠物工坊和外观导入只在云灵工作区集中显示；
- 切换工作区后这些设置不会遮挡 Memory、Audit 或 Network Lab；
- 移动后的按钮、输入框、试听、导入和开关仍可操作；
- 修改设置后刷新 Side Panel，确认本地保存结果不变。

## 安全与无干扰

- Audit 生成补丁仍要求现有 Local Agent 连接和令牌；
- Network Lab 规则、Mock 和已有请求分析不受影响；
- 多次打开和关闭 Side Panel 不累积标签页、运行时消息、存储或 MutationObserver 监听器；
- DevTools 中确认没有定时轮询、额外上传请求或新增权限；
- 窄 Side Panel、键盘 Tab、Enter、空格和减少动画模式均可使用。
