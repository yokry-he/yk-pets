# Pet Studio 高级交互

## 搜索与定位

顶部搜索框支持按部位、参数和常用名称定位。输入“鼻子”“肚皮”“颜色”“尾巴”“轨道”或英文关键词后，选择结果会切换到对应工作区。非输入状态下按 `/` 聚焦搜索，按 `Escape` 清空搜索并退出经典对比。

## 当前与经典外观对比

预览工具栏提供“对比经典”。进入后保存当前配方快照，把预览和控件临时切换为扩展经典配方，并锁定编辑和保存操作。返回当前外观或关闭页面时会恢复精确快照，不写入草稿或撤销历史。界面显示当前外观与经典外观不同的配置组。

## 点击式部位聚焦

预览上方提供头部、身体、尾巴和标志热点。热点支持鼠标和键盘，点击后切换到对应编辑区。热点不读取宿主网页，也不创建新的 WebGL 拾取循环或第二个渲染场景。

## 本地方案库与历史

高级面板可以为当前外观命名并保存到本机。方案可应用或删除，继续复用现有 Pinia Store 和 `localStorage`，不上传。面板显示最近自动草稿时间、撤销条数和重做条数。
