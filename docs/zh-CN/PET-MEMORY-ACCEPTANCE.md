# 宠物记忆人工验收清单

自动测试无法替代 Chrome Side Panel、右键菜单、快捷键和网页内 3D 反馈的真实浏览器检查。发布前至少完成以下流程。

## 快速记录

- 打开一个普通 HTTP/HTTPS 页面；
- 使用 `Ctrl/Command + Shift + .`；
- Side Panel 打开并切换到“宠物记忆”；
- 编辑器带入页面标题和 URL，并获得键盘焦点；
- 输入正文后按 `Ctrl/Command + Enter`；
- 新卡片进入收件箱，网页宠物显示保存反馈。

## 选中文字

- 在网页中选中一段文字；
- 右键选择“交给云灵记住”；
- 不打开大型遮挡层；
- 收件箱生成一张网页摘录卡；
- 当前页面的宠物记忆徽章数量增加；
- 刷新页面后徽章数量仍正确。

## 状态与编辑

- 收件箱卡片可以加入待办；
- 待办可以标记完成；
- 完成时云灵产生庆祝反馈；
- 已完成卡片可以恢复为待办；
- 编辑标题、正文、标签和优先级后保存；
- 归档后在 6 秒内撤销；
- 搜索和标签筛选结果正确。

## 页面审计集成

- 执行一次页面审计；
- 在问题卡点击“记住”；
- 自动切换到宠物记忆；
- 生成的卡片处于待办状态；
- 标题、说明、页面、selector、类别标签和优先级正确。

## 导出

- Markdown 文件可下载且内容可读；
- JSON 文件可下载且结构完整；
- 归档内容不进入默认导出。

## 性能与无干扰

- Side Panel 关闭时，宠物记忆不会持续扫描网页 DOM；
- 页面只监听扩展本地存储变化，不轮询；
- 保存或完成反馈不会打断正在进行的手动动作；
- 页面记忆徽章不会遮挡审计问题徽章或宠物点击区域。
