# 宠物记忆人工验收清单

自动测试无法替代 Chrome Side Panel、右键菜单、快捷键和网页内 3D 反馈的真实浏览器检查。发布前至少完成以下流程。

## 快速记录

- 打开一个普通 HTTP/HTTPS 页面；
- 使用 `Ctrl/Command + Shift + .`；
- Side Panel 打开并切换到“宠物记忆”；
- 编辑器带入页面标题和 URL，并获得键盘焦点；
- 输入正文后按 `Ctrl/Command + Enter`；
- 新卡片进入收件箱，网页宠物显示保存反馈。

## 选中文字

- 在网页中选中一段文字；
- 右键选择“交给云灵记住”；
- 不打开大型遮挡层；
- 收件箱生成一张网页摘录卡；
- 当前页面的宠物记忆徽章数量增加；
- 刷新页面后徽章数量仍正确。

## 当前页面记忆徽章

- 页面有未归档记忆时，徽章显示正确数量；没有记忆时不显示增强按钮；
- 使用鼠标点击徽章，确认只打开 Side Panel，不同时打开宠物环形菜单或开始拖动；
- 使用 `Tab` 聚焦徽章，并分别用 `Enter` 和空格触发；焦点轮廓清晰；
- 预先设置搜索词和标签筛选，再点击徽章，确认两者被清除；
- Side Panel 直接激活“当前页面”视图，并把焦点放在该筛选按钮；
- Side Panel 已停留在页面审计或网络实验室时，点击徽章仍会切换到宠物记忆；
- 窄 Side Panel 中加载、成功或失败状态不遮挡主要操作；
- 点击后立即切换活动标签页，确认请求不会应用到错误页面，并显示重试提示；
- 请求完成或超过 30 秒后，本地一次性请求键被删除；
- 重复更新徽章数量不会累积按钮、MutationObserver 或存储监听器。

## 状态与编辑

- 收件箱卡片可以加入待办；
- 待办可以标记完成；
- 完成时云灵产生庆祝反馈；
- 已完成卡片可以恢复为待办；
- 编辑标题、正文、标签和优先级后保存；
- 归档后在 6 秒内撤销；
- 搜索和标签筛选结果正确。

## 页面摘录重新定位

- 在当前页面保存一段跨多个内联元素的文字摘录；
- 在记忆卡点击“定位摘录”，确认页面滚动到原文并出现精确选区与短时高亮；
- 页面有多处相同文字时，确认结果提示匹配数量并定位第一处可见文本；
- 切换到其他标签页后再次点击，确认新标签页打开来源页面并完成定位；
- 修改或删除来源文字后点击，确认显示明确失败状态且不会持续扫描；
- 分别验证带 selector 的导入卡片和没有 selector 的旧卡片，确认两者都能定位或给出明确失败；
- 高亮约 12 秒后自动收起，`prefers-reduced-motion` 下不使用平滑滚动；
- 重复点击定位不会持续增加 scroll、resize 或 runtime 监听器。

## 页面审计集成

- 执行一次页面审计；
- 在问题卡点击“记住”；
- 自动切换到宠物记忆；
- 生成的卡片处于待办状态；
- 标题、说明、页面、selector、类别标签和优先级正确。

## JSON 导入与导出

- Markdown 文件可下载且内容可读；
- JSON 文件可下载且结构完整；
- 归档内容不进入默认导出；
- 从导出菜单选择“导入 JSON”，重新导入刚导出的文件；
- 导入过程中按钮显示加载状态，完成后显示导入、重复、冲突、无效和容量截断数量；
- 首次导入新增卡片，再次导入同一文件只报告重复，不产生副本；
- 修改导入文件中一个已有 ID 的内容后再次导入，该卡片报告冲突且不会覆盖本地内容；
- 导入不含 `cards` 数组、版本不是 1、JSON 语法错误或超过 8 MB 的文件时显示明确错误；
- 导入后当前页面筛选、搜索、状态操作和宠物记忆徽章立即更新；
- 导入文件不会触发网络上传或新增权限。

## 性能与无干扰

- Side Panel 关闭时，宠物记忆不会持续扫描网页 DOM；
- 页面只监听扩展本地存储变化，不轮询；
- 保存、导入或完成反馈不会打断正在进行的手动动作；
- 页面记忆徽章不会遮挡审计问题徽章或宠物点击区域。
