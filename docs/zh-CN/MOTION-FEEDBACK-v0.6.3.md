# NOVA v0.6.3：新增规则、动作音效与思考气泡

## 新增规则修复

- 规则编辑器构建数据前会先解除 Vue Proxy，避免 `structuredClone` 抛出 `DataCloneError`。
- “新增规则”和“基于请求生成规则”已通过真实编辑器状态机的可执行回归测试。
- 新建流程不再静默自动恢复旧草稿；草稿只能由用户点击恢复入口加载。
- 草稿入库结构校验更严格，规则名称、站点范围、URL、方法和动作类型都必须完整。
- 编辑器子页报错时会显示错误边界与返回按钮，不再留下空白工作区。
- 移除 TresJS ready 回调中对非渲染器对象调用 `setClearColor` 的不安全逻辑。

## 动作交互

- 重复点击正在执行或正在过渡到的同一动作时，不重启动画、不重播音效。
- 点击不同动作会打断当前动作，经过 240ms 中性姿态后开始新动作。
- 鼠标悬停只记录视线意图，不启动、打断或排队任何行为；动作进行时同时锁定视线输入。
- 所有保留动作的时长均适当增加，云朵午睡改为平躺并延长到 18 秒。

## 音效与思考气泡

- 18 个保留动作均配置独立的音高序列，使用 Web Audio 在本地生成，不需要远程媒体资源。
- 首次音效仅在用户主动点击动作后创建音频上下文，遵守浏览器自动播放限制。
- 动作开始时，宠物头部会出现包含动作名称和专属中文文案的思考气泡。

## 验证

- v0.6.3 专项静态回归：16/16。
- 新增规则真实状态机回归：手动新增与请求生成两条路径均通过。
- Extension Vue TypeScript 检查通过。
