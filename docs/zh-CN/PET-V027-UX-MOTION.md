# NOVA 宠物交互优化（v0.2.7）

## 本次实现范围

本版本按照“推荐方案”先落地中文单语版本，重点优化以下内容：

1. **动作与功能按钮样式可读性**
   - 按钮仅保留中文标签。
   - 提高文字、边框、背景对比度。
   - 缩短环形按钮与宠物的距离，形成更紧凑的半环布局。

2. **宠物身体结构优化**
   - 将前爪整体上移到胸口下方，避免看起来像只有两条腿。
   - 增加后爪结构，让角色能完整展示四肢。
   - 调整相机参数，保证完整身体更容易显示在可视区域中。

3. **尾巴与整体视觉优化**
   - 保留分段柔性云尾方向。
   - 调整尾根位置，让尾巴连接更自然。

4. **底座改为动态阴影**
   - 删除实体底座和发光圆盘。
   - 改为软接触阴影。
   - 跳跃时阴影按高度缩放与减淡，落地时恢复。

5. **动画结束过渡优化**
   - 修复转圈后明显反向回到初始朝向的问题。
   - 使用角度包裹方式，使动作结束后朝向自然回稳。

6. **闲时轮播与直接动作控制**
   - 保留点击动作按钮直接触发动作。
   - 保留空闲时自动动作轮播。

## 当前已知限制

1. 本次仍基于程序化几何体模型，不是完整骨骼动画模型。
2. 后续若要继续提升动作表现，建议下一阶段拆出更明确的 Position Rig / Action Rig / Look Rig。
3. 当前先使用中文单语，后续再补多语言资源表与 i18n 切换。

## 下一步建议

1. 为动作按钮增加 Tooltip 和悬停预备动作。
2. 进一步收紧环形菜单布局，在窄屏下切换为紧凑网格。
3. 将阴影参数和动作参数抽出为配置项，便于后续调优。
4. 若追求更高质量肢体动作，建议后续替换为 GLB 狐狸模型并接入状态机动画。
