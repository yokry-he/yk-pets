# 宠物可配置系统

## 统一配方与唯一模型

YK-PETS 使用一个 `customization` 配方层和一个正式 `ExtensionAlignedCloudFox.vue` 组合层。旧版 Studio、扩展经典外观和导入 JSON 会先经过兼容归一化，再进入同一套模型、动作和 Shape Profile；不会为默认、导入或 Studio 外观复制第二套宠物。

## Studio 稳定布局

Studio 的搜索、经典对比、部位热点、最近修改和本地方案库已经直接集成到 Vue 页面，不再由客户端插件在挂载后移动或插入 DOM。

- 顶部使用明确的标题、搜索和操作三列 Grid；
- 宽度不足时按固定断点换行；
- 检查器的高级面板和参数区拥有独立布局与滚动；
- 主 Studio 页面隐藏固定跨页入口和网页宠物覆盖层，避免遮挡；
- `/` 聚焦搜索，`Escape` 清空搜索并退出只读经典对比。

## 唯一面部层级

头部现在只有一个动画 Head Rig 和一个 FaceRoot：

- 正式头部不再同时绘制旧鼻嘴、遮挡椭球和新鼻嘴；
- 鼻子、嘴巴、舌头和脸颊直接使用头部局部坐标；
- 头部动作只计算一次，面部随唯一 Head Rig 一起运动；
- 默认 `smile` 恢复为原正式云狐的经典圆润嘴和舌头；
- 猫系、线条、张嘴和嘟嘴使用薄曲线或薄口腔几何，避免侧面悬空。

鼻子保留柔软圆鼻、狐狸三角鼻、感应器鼻、纽扣鼻和爱心鼻。口鼻部、鼻子、嘴巴、舌头与脸颊可分别着色。

## 眼睛

星芒眼与水晶菱眼不再共享同一个十二面体：

- 星芒眼由交叉细长光片构成明显星形；
- 水晶菱眼使用纵向八面晶体和分面材质；
- 圆润眼和椭圆眼继续支持球、网页指针和烟花追视；
- 星芒、菱眼和目镜不再叠加圆形追视覆盖层。

## 六种身体和匹配头型

共享 `cloud-fox-shape-profile.ts` 同时驱动身体外轮廓、头型、眼耳间距、口鼻深度、肚皮位置和镜头边界。

- 球体：均匀圆身体和圆头；
- 椭圆体：扩展经典椭球与经典头；
- 胶囊体：中央圆柱和圆润端部，配较长头型；
- 梨形：下宽上窄身体，配紧凑圆头；
- 豆形：偏斜双瓣轮廓，配轻微倾斜椭圆头；
- 圆角方糖：`RoundedBoxGeometry` 身体和匹配的圆角方形头。

新外轮廓包覆正式身体基线，因此原前爪、后肢、尾巴、标志和三十个动作继续使用原实现。Shape Profile 还为各身体提供头部偏移、面部前移量、肚皮正面深度和边界倍率。

## 肚皮

默认肚皮为显式**椭圆形**。可选椭圆形、蛋形、盾牌形、水滴形、倒水滴形、不对称豆形、圆角矩形、爱心形、云朵形和胸毛形。

肚皮支持显示、颜色、宽高、左右/上下位置、旋转和边缘柔和度。肚皮现在按 Shape Profile 使用正面贴片，不再把旧椭球薄壳强行套在胶囊、梨形、豆形或圆角方糖上。

## 全部位颜色与扩展范围

颜色工作区覆盖身体、四肢、爪、口鼻部、鼻、嘴、舌、脸颊、眼睛、高光、耳朵、触角、肚皮、尾尖发光、能量核心、轨道和标志。现有单模型中的身体/尾根、四肢/尾中段和爪部/尾尖暖色关联会在 UI 中明确标注。

Studio 保留扩展创作范围，并由“检查”工作区提示潜在穿模，不静默截断。配方和方案仅保存在本机，不上传，也不新增浏览器权限或后台轮询。
