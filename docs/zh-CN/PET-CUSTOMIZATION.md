# 宠物可配置系统

## 本批次范围

本阶段为 YK-PETS 云灵建立统一的 `customization` 配方层。旧版 Studio、扩展经典外观和导入 JSON 会先经过兼容归一化，再进入同一套 Cloud Fox 渲染器。

## 鼻子与嘴巴

鼻子和嘴巴不再只是写入配方的无效下拉选项。统一头部现在根据选项渲染明显不同的几何：

- 鼻子：柔软圆鼻、狐狸三角鼻、感应器鼻、纽扣鼻、爱心鼻；
- 嘴巴：微笑嘴、猫系嘴、安静线条嘴、开心张嘴、软萌嘟嘴；
- 说话、吃饭、开心、喷嚏等动作继续驱动嘴部开合；
- 鼻子、嘴巴、舌头、口鼻部毛色和脸颊分别可着色。

## 肚皮

默认肚皮改为显式的**椭圆形**，不再通过“模型默认”暗中绑定盾牌形。

可选轮廓：

1. 椭圆形；
2. 蛋形；
3. 盾牌形；
4. 水滴形；
5. 倒水滴形；
6. 不对称豆形；
7. 圆角矩形；
8. 爱心形；
9. 云朵形；
10. 胸毛形。

肚皮支持显示开关、颜色、宽度、高度、左右位置、上下位置、旋转和边缘柔和度。旧版 `model-default` 会迁移为椭圆形；旧版自定义轮廓会尽量映射到对应新名称。

## 全部位颜色

Studio 的“颜色”工作区覆盖当前统一模型的全部可见材质通道：

- 身体主体、四肢、爪部；
- 口鼻部毛色、鼻子、嘴巴、舌头、脸颊；
- 眼睛与高光；
- 外耳、内耳、耳尖；
- 触角杆与触角尖；
- 肚皮、尾尖发光、能量核心；
- 主副轨道、胸口标志和后背标志。

为了兼容现有唯一模型，身体主体同时控制尾根材质，四肢同时控制尾中段，爪部同时控制尾尖暖色和触角杆；这些关联在 UI 中明确标注，不会伪装成独立通道。

## 参数范围

Studio 使用扩展创作范围，不再把原安全范围当成硬限制。主要范围包括：

- 身体宽度约 `0.55–1.70`；
- 身体高度约 `0.55–1.72`；
- 头部大小约 `0.68–1.48`；
- 四肢长度约 `0.42–1.38`；
- 耳朵和眼睛大小约 `0.52–1.72`；
- 尾巴长度约 `0.50–1.90`；
- 触角比例约 `0.42–1.82`；
- 肚皮宽高约 `0.42–1.72`。

超出旧推荐范围时由“检查”工作区提示潜在穿模，而不是静默截断用户输入。

## Studio 交互

- 部件使用卡片选择，鼻嘴和身体形状不再只用文字下拉框；
- 滑杆旁提供精确数值输入和单项重置；
- 一次拖动只生成一条撤销历史；
- 编辑草稿自动保存在本地；
- “保存并用于同步”明确表示写入正式 Studio 外观；
- 导入旧 JSON 时自动补齐颜色、肚皮和扩展范围字段；
- 不上传配方，不新增浏览器权限。
