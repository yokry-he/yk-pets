# 宠物可配置系统人工验收

自动 CI 不能替代真实 Chrome、WebGL、不同缩放比例、四视角和动作视觉检查。

## Studio 布局

- 在 1280、1440、1658 和 1920 宽度下打开 `/studio`，确认标题、搜索和操作按钮不重叠、不漂浮出顶部容器；
- 使用 80%、100% 和 125% 浏览器缩放，确认左侧导航、中间预览和右侧检查器保持可用；
- 展开“方案与最近修改”，确认参数区仍能独立滚动到底部；
- 确认主 Studio 页面没有右下角固定“预设与风格”按钮，也没有网页宠物覆盖编辑器；
- 检查搜索、经典对比、热点、方案保存/应用/删除、`/` 和 `Escape`。

## 唯一鼻嘴层级

- 在正面、左侧和右侧逐一检查五种鼻子；鼻子应贴在口鼻部表面，不与嘴重叠或悬空；
- 默认微笑应恢复原正式云狐的经典圆润嘴和舌头；
- 检查猫系、线条、张嘴和嘟嘴，侧面不应看到垂直吊起的粗圆环；
- 播放 Idle、Talking、Eating、Happy 和 Sparkle Sneeze，确认鼻嘴只随一个头部动作空间移动；
- 改变口鼻部、鼻、嘴、舌和脸颊颜色，确认没有旧部件从下方露出。

## 眼睛

- 星芒眼应呈交叉星形光片，水晶菱眼应呈纵向分面晶体；两者不得显示相同轮廓；
- 圆润眼和椭圆眼在网页指针、玩球、飞扑和烟花动作中继续追视；
- 星芒、菱眼和能量目镜不应出现额外圆形追视覆盖层；
- 睡眠、眨眼、困惑和兴奋动作仍应改变眼睛状态。

## 六种身体和头型

- 依次选择球体、椭圆体、胶囊体、梨形、豆形和圆角方糖，正面和侧面轮廓都应可直接区分；
- 球体应接近均匀圆形，胶囊体应有直中段和圆端，梨形应下宽上窄，豆形应有偏斜双瓣；
- 圆角方糖身体和头部都应使用圆角方形，不应保留明显球形头；
- 检查耳朵、眼睛、口鼻部、前爪、后肢、肚皮和尾根在每种形状上的连接；
- 播放全部常用动作，确认新外轮廓没有遮掉四肢或破坏尾巴；
- 检查四视角和自动镜头，宠物不应被裁切。

## 肚皮、颜色和范围

- 默认肚皮为椭圆形，并逐一检查十种名称与轮廓；
- 在六种身体上检查肚皮位置、颜色、宽高、旋转和柔和度，侧面不应出现旧椭球薄壳；
- 修改所有颜色通道并确认 Studio 和 Chrome 扩展一致；
- 将身体、头部、四肢、耳眼、尾巴、触角和肚皮调整到扩展范围边界，检查页应提示风险而不是静默回滚；
- 保存、刷新、导出和重新导入后，所有配置应保留。

## 安全边界

- 确认没有新增 Chrome 权限、数据上传或后台轮询；
- 确认 Network Lab、Pet Memory、Page Audit 和 Local Agent 工作流保持可用；
- 关闭 3D 后仍应销毁 WebGL，重新开启后恢复当前正式配方。
