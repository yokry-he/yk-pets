# 云狐工坊

## 1. 目标

云狐工坊是 YK-PETS 的首个宠物外观创作工具。它以物种模板为约束，让用户在不破坏动作骨架和运行性能的前提下创建云狐个体。

当前默认个体：

- 名字：云灵；
- 英文名：Zeph；
- 宠物 ID：`zeph`；
- 物种：云狐；
- 物种 ID：`cloud-fox`。

产品品牌、宠物名字和物种是三个独立概念。修改外观不会修改个体身份、亲密度或后续记忆数据。

## 2. 当前入口

开发环境启动 Playground：

```bash
pnpm dev:playground
```

打开：

```text
http://localhost:3000/studio
```

Playground 其它页面右下角也提供“云狐工坊”入口。

## 3. 外观配方

外观通过 `CloudFoxAppearanceRecipe` 保存，而不是保存一份完整的 Three.js 场景。配方包括：

- 身份：宠物 ID、中英文名、名字缩写；
- 部件：耳朵、眼睛、鼻子、嘴、尾巴、触角；
- 比例：身体、头部、前肢、爪子、眼睛、耳朵、尾巴和触角；
- 颜色：主毛色、阴影毛色、腹部、眼睛和多个发光通道；
- 发光：固定颜色、动作状态联动和彩虹循环；
- 标志：胸口文字、后背文字、大小与发光颜色。

配方当前使用：

```text
schemaVersion: 1
speciesId: cloud-fox
```

导入旧或不完整的 JSON 时会经过归一化，未知部件会回退到默认值，比例会被限制到物种安全范围。

## 4. 部件槽位

### 4.1 耳朵

- 星云尖耳；
- 柔软圆耳；
- 羽翼耳；
- 机械耳。

### 4.2 眼睛

- 圆润眼；
- 灵动椭圆眼；
- 星芒眼；

### 4.3 鼻子与嘴

鼻子支持柔软鼻、狐狸三角鼻和感应器鼻。嘴支持微笑嘴、猫系嘴和安静线条嘴。

### 4.5 触角

- 无触角；
- 双星触角；
- 弧光触角；
- 晶体触角。

## 5. 比例安全范围

四肢和身体不允许自由拖动骨骼，而使用受限参数。当前安全范围包括：

| 参数 | 最小值 | 最大值 |
|---|---:|---:|
| 身体大小 | 0.82 | 1.18 |
| 头部大小 | 0.84 | 1.20 |
| 前肢长度 | 0.78 | 1.22 |
| 前肢间距 | 0.82 | 1.20 |
| 爪子大小 | 0.76 | 1.32 |
| 耳朵大小 | 0.76 | 1.28 |
| 眼睛大小 | 0.76 | 1.32 |
| 眼睛间距 | 0.82 | 1.18 |
| 尾巴长度 | 0.78 | 1.34 |
| 尾巴宽度 | 0.76 | 1.32 |
| 触角大小 | 0.72 | 1.36 |

这些范围属于物种定义，未来其它物种可以声明不同的可编辑能力与范围。

## 6. 发光通道

当前配方拥有：

- `primaryGlow`：主发光；
- `secondaryGlow`：辅助发光；
- `tailGlow`：尾巴；
- `antennaGlow`：触角；
- `symbolGlow`：身份标志。

发光模式：

- `fixed`：使用用户指定颜色；
- `emotion`：根据待机、招手、跳跃、伸展、转圈和趴下状态切换；
- `rainbow`：运行时循环色相。

不使用全屏 Bloom 后处理，避免宠物注入普通网页后显著增加渲染成本。

## 7. 胸口和后背标志

默认：

- 胸口：英文名首字母 `Z`；
- 后背：品牌简写 `YK`。

文字通过 Canvas 动态生成透明纹理，再贴到身体预设曲面。这样可以即时修改 1–3 个字符，不需要生成新的字体几何体。

## 8. 编辑与验证

工坊支持：

- 正面、左侧、背面和右侧视角；
- 深色、浅色和模拟网页背景；
- 待机、招手、跳跃、伸展、转圈和趴下动作测试；
- 浏览器本地保存；
- 恢复默认；
- 随机生成；
- JSON 导入和导出。

调整四肢、尾巴和触角后，应至少播放全部动作测试并检查：

- 四肢是否进入身体；
- 爪子是否离地或明显穿插；
- 尾巴是否穿过身体和平台；
- 触角是否进入耳朵；
- 胸背标志在对应视角是否清晰。

## 9. 当前边界

本阶段是云狐工坊 MVP：

- 渲染器位于 Playground，尚未替换扩展正式云狐；
- 暂不支持任意 GLB 导入；
- 暂不支持自由骨骼拖动；
- 暂不支持多尾和左右非对称编辑；
- 暂不支持社区部件包。

后续将把外观配方接入扩展运行时，并提取为框架无关的宠物定义和渲染适配器。
