# NOVA v0.5.2: Bilingual Documentation and Maintainability

v0.5.2 keeps runtime behavior unchanged and improves delivery, release, onboarding, and maintenance quality.

It adds detailed Chinese and English user, Network Lab, build/release, troubleshooting, maintenance, and comment-standard guides, plus English current-feature documents for v0.5.0 and v0.5.1.

Every handwritten TypeScript, Vue, and CSS file now has a bilingual responsibility header. Large core files also document state groups, lifecycle, matching, cross-world messaging, animation phases, template regions, and CSS sections.

Documentation completeness and source-header validation are added to the quality gates alongside existing bilingual, type, domain, test, and build checks.
