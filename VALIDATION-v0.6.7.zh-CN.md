# NOVA Browser Agent v0.6.7 验证报告

## 核心场景

- 工作区导航不会进入或残留任务忙碌状态，3D 加载光环只在真实任务期间出现。
- 页面审计按五类多选范围执行，未勾选规则从执行入口跳过，报告保存实际范围。
- Network 请求可按 DevTools 风格的资源类型分类、计数和组合筛选。
- 动作文案使用不可交互的小气泡，18 条手动动作使用内置中文语音且不再播放提示音。
- 尾尖保持明亮常亮，相关动作期间连续进行多色变化。

## 已知构建提示

Playground 的 Three.js/TresJS 客户端 Chunk 仍超过 500 KB。该提示与以前版本一致，不影响扩展构建、页面审计 Content Script 或本轮功能。

## 安装后人工确认

自动检查无法替代真实 Chrome 页面交互。重新加载 v0.6.7 并刷新目标网页后，建议按以下顺序确认：

1. 打开网络实验室，确认宠物不出现永久加载圈。
2. 分别只勾选一个审计类型运行审计，确认报告范围和问题类别一致。
3. 在有图片、脚本、接口与 WebSocket 的页面检查请求分类。
4. 连续切换两个动作，确认小气泡不挡操作、旧语音停止且没有提示音。
5. 在深色页面检查尾尖常亮，并触发尾光流动或甩尾旋风确认多色闪烁。
