# NOVA 极简菜单设计（v0.3.3）

## 目标

将原来的三段文字切换器压缩为一个当前模式图标。点击该图标时展开“功能、动作、工程”三个模式图标，选择后自动收回。菜单按钮继续使用纯图标与中文 Tooltip。

## 交互规则

1. 点击宠物打开菜单时，仅显示当前模式图标和当前模式的六槽位按钮。
2. 点击当前模式图标展开三个模式图标。
3. 选择模式后自动收回为单图标，并切换对应按钮。
4. 再次点击宠物、点击菜单外部或按 `Esc` 关闭整个菜单。
5. 审计思考气泡独立于菜单，不会随菜单关闭而消失。
6. 上一次使用的模式存入扩展本地存储，下次打开继续使用。

## 固定六槽位

功能、动作和工程共用同一套六槽位，从左下角开始依次填充：

1. 左下
2. 左侧偏下
3. 左侧中部
4. 左上
5. 上方偏左
6. 上方偏右

不足六个时剩余槽位为空；超过六个时分页，每页最多六个。

## Tooltip

所有模式和菜单按钮默认只显示图标。鼠标悬停或键盘聚焦时，使用独立顶层 Tooltip 显示中文名称与说明。点击按钮后 Tooltip 立即关闭。

## 可扩展性

功能、动作和工程菜单均由注册表驱动。后续新增动作只需追加定义；第七项开始自动进入下一页，无需新增固定 CSS 坐标。
