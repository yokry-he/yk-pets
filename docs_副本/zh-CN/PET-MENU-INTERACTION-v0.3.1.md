# NOVA 宠物菜单交互优化（v0.3.1）

## 目标

本版本先采用中文单语界面，统一宠物菜单的显示、隐藏和模式切换行为，并减少动作按钮的视觉噪声。

## 已确认交互

### 点击宠物

- 菜单关闭时：同时显示“功能 / 动作”切换器、关闭按钮和当前模式对应的环形按钮。
- 菜单打开时：再次点击宠物会同时隐藏切换器、关闭按钮、环形按钮、工程工具面板和动作提示。
- 审计思考气泡独立存在，不随菜单关闭而自动消失。

### 功能与动作切换

- 首次打开默认进入“功能”。
- 后续重新打开时保留上一次选择的模式。
- 切换模式时，旧按钮组消失，新按钮组同步出现，不允许两组同时残留。

### 动作按钮

- 默认只显示图标。
- 鼠标悬停或键盘聚焦时显示中文动作说明。
- 点击后直接播放动作，不再显示额外的动作名称胶囊。
- 动作失败时才使用思考气泡显示错误信息。

### 布局

- “功能 / 动作”切换器固定锚定在宠物上方，不再绑定提示气泡的位置。
- 切换器与关闭按钮处于同一水平区域。
- 动作按钮缩小并靠近宠物，形成紧凑的左上半环。
- 窄屏下改为紧凑网格布局。

## 后续事项

- 当前先使用中文单语，后续再统一接入多语言资源表。
- 后续可增加按钮悬停时的宠物视线跟随和轻微预备动作。
