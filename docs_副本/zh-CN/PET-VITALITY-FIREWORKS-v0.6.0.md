# NOVA v0.6.0：宠物生命力、烟花与随机彩蛋

## 1. 版本目标

v0.6.0 将 NOVA 从“动作菜单驱动的 3D 宠物”升级为具有生活状态、精选表演动作和低概率随机彩蛋的数字伙伴。新增能力仍遵循以下原则：

- 不阻挡网页正文和主要操作；
- 高能效果由用户主动触发为主；
- 自动彩蛋具有独立长冷却和低触发概率；
- 页面后台、菜单打开、Agent 忙碌或减少动态效果模式下暂停自动播放；
- 所有动作仍通过动作注册表和统一行为协议驱动。

## 2. 尾巴尖与触角

### 2.1 尾巴尖发光

尾巴尖由三层效果组成：

1. 发光尾段：提高基础自发光强度；
2. 能量核心：球形核心持续进行柔和呼吸脉冲；
3. 外层光晕与星点：动作强度越高，拖尾与星点越明显。

待机时保持克制的持续光，高能动作、烟花和能量爆发时提高亮度。休息和云朵午睡时自动降低粒子密度。

### 2.2 双触角

头部新增左右两个弧形能量触角。触角根部跟随头部，末端为发光能量珠。

- 待机：低幅度弹性摆动；
- 思考：规律脉冲；
- 星星杂耍：中等亮度；
- 星光喷嚏：先充能再瞬时爆亮；
- 高空烟花：发射阶段持续高亮；
- 能量爆发：触角、尾尖和胸口核心同步聚能。

## 3. 新增动作

### 3.1 害羞偷看

动作过程：

1. 身体轻微侧倾；
2. 两只前爪靠近脸颊；
3. 脸红增强；
4. 头部歪向一侧并短暂偷看用户；
5. 平滑恢复待机。

该动作属于普通闲时动作，也可以在动作菜单中手动触发。

### 3.2 星星杂耍

三颗能量星在左右前爪和尾尖附近循环抛接。宠物的眼睛与头部会跟随当前位于最高点的星星，左右前臂交替配合。

动作持续约 6.4 秒，属于低频生活动作。

### 3.3 云朵午睡

宠物召唤一朵柔和发光云：

- 身体降低并放松；
- 前爪和后爪收起；
- 眼睛闭合；
- 触角变为低频呼吸光；
- 云朵和睡眠符号缓慢漂浮。

动作持续约 7.6 秒，默认只低频自动播放或手动触发。

### 3.4 星光喷嚏

触角先快速聚能，宠物做出短暂忍住喷嚏的表情，然后从鼻尖释放少量星光。该动作时长短、粒子数量低，是随机彩蛋池的一部分。

### 3.5 高空烟花秀

烟花动作包含三次连续发射：

1. 宠物轮流举起前爪；
2. 能量弹从宠物附近升到画布高处；
3. 在高空爆炸；
4. 宠物始终抬头观看；
5. 触角、尾尖与胸口核心同步增强。

每次爆炸从以下精选形状中确定一种：

- 菊花/球形；
- 环形；
- 爱心；
- 星形。

颜色只从预设的蓝紫、金粉、薄荷青和粉紫色板中选择，避免完全随机导致颜色混乱。烟花动作会暂时扩大相机视野，使爆炸位置明显高于宠物头部。

## 4. 随机彩蛋调度

自动动作现在分为四层：

| 层级 | 大致间隔 | 内容 |
|---|---:|---|
| normal | 25–45 秒 | 招手、跳跃、害羞偷看等 |
| rare | 60–120 秒 | 玩球、吃饭、星星杂耍、云朵午睡 |
| high | 180–300 秒，额外概率门槛 | 空翻、甩尾旋风、飞扑接球、能量爆发 |
| easter | 240–420 秒，约 28% 通过概率 | 星光喷嚏、高空烟花秀 |

任何一层动作播放后都会重新进入冷却。相同动作不会连续作为自动动作触发。

## 5. 减少动态效果

当系统启用 `prefers-reduced-motion: reduce` 时：

- 烟花扩散半径降低；
- 高能位移和旋转幅度下降；
- 自动动作调度停止；
- 用户仍可在动作菜单中选择动作，但呈现为简化版本。

## 6. 扩展新动作

新增动作必须同步更新：

1. `packages/shared/src/messages.ts` 中的 `NovaPetBehavior`；
2. `apps/extension/components/avatar/pet-motions.ts` 注册表；
3. `CloudFox.vue` 的阶段时长、行为标志、Rig 动画和视觉道具；
4. 自动动作层级和中英文文档；
5. v0.6.0 专项检查脚本。
