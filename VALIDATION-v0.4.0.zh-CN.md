## 未完成检查

当前运行环境持续无法连接 npm registry：

```text
EAI_AGAIN registry.npmjs.org
UND_ERR_CONNECT_TIMEOUT
```

因此未能从空依赖目录执行：

```bash
pnpm install
pnpm typecheck
pnpm build:extension
pnpm zip:extension
```

本次不会提供声称已经重新编译的 Chrome 生产包。源码包可在网络正常的本地环境执行：

```bash
corepack enable
pnpm install
pnpm typecheck
pnpm build:extension
pnpm zip:extension
```

## 建议人工验收

构建后重点检查：

1. 前爪是否位于胸口两侧，后爪是否清晰可见。
2. 尾尖是否始终在身体轮廓之外。
3. 玩球时球是否按弧线弹跳，前爪是否交替拍球。
4. 吃饭时头部、嘴巴和食物减少是否同步。
5. 伸懒腰结束是否自然恢复。
6. 转圈结束是否不发生反向回正。
7. 跳跃时阴影是否缩小、变淡，落地时是否恢复。
8. 动作菜单是否因 9 个动作自动出现第二页。
