# NOVA Browser Agent v0.6.0 Release Notes / 发布说明

## 中文

v0.6.0 是宠物生命力与网络实验室可用性升级版本。

### 3D 宠物
- 尾巴尖发光增强；
- 新增双发光触角与聚能状态；
- 新增高空烟花秀，包含三次连续发射和四种精选花型；
- 新增害羞偷看、星星杂耍、云朵午睡和星光喷嚏；
- 新增独立 `easter` 闲时层级和长冷却随机彩蛋机制；
- 烟花时自动扩展相机视野；
- 支持减少动态效果降级。

### 网络实验室
- 修复直接新增规则和从请求生成规则后的页面同步链路；
- 请求列表支持逐条展开；
- 详情新增 Query、请求头、响应头、请求体、响应预览和 Timing；
- Fetch/XHR 采集增加请求头、响应头和请求体预览；
- 敏感头和敏感字段继续脱敏。

## English

v0.6.0 improves both pet vitality and Network Lab usability.

### 3D pet
- Brighter tail-tip glow.
- Dual emissive antennae with energy charging.
- High-altitude three-launch fireworks show with four curated shapes.
- New shy peek, star juggling, cloud nap, and sparkle sneeze motions.
- Separate long-cooldown `easter` idle tier.
- Wider camera framing during fireworks.
- Reduced-motion fallback.

### Network Lab
- Fixed page synchronization after manual rule creation and request-based rule generation.
- Per-request expandable details.
- Query, request/response headers, request body, response preview, and timing details.
- Richer Fetch/XHR capture with continued sensitive-data redaction.
