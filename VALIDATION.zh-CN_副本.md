# NOVA Browser Agent v0.2.7 验证报告

## 本次范围

v0.2.7 先采用中文单语界面，主要实现：

- 功能和动作按钮仅显示中文；
- 提高按钮文字、背景和边框对比度；
- 缩短环形按钮与宠物之间的距离；
- 前爪上移到胸口下方；
- 增加左右后爪；
- 删除实体底座和发光圆环；
- 增加柔和接触阴影；
- 跳跃时阴影缩小、变淡，落地时恢复并短暂扩张；
- 调整相机以展示完整身体；
- 修复转圈结束后明显反向回正；
- 闲时完整动作改为约 25～45 秒低频触发。

## 环境限制

当前环境无法解析 `registry.npmjs.org`，`pnpm install` 返回 `EAI_AGAIN`，因此未能从空依赖目录重新运行完整 WXT 构建。

本次可加载 Chrome 包基于已验证的 v0.2.6 生产构建进行确定性修改：每一个生产代码替换点均设置了唯一匹配断言，替换失败时不会生成包。完整 Vue/WXT v0.2.7 实现包含在源码包中。

此外，容器中的 Chromium 无法初始化 EGL/WebGL，因此未完成像素级 3D 截图验证。建议在本地 Chrome 中重新加载扩展并刷新目标网页后，重点检查：

1. 四只爪子的前后层次；
2. 阴影在跳跃过程中的大小和透明度；
3. 转圈结束是否自然保持朝向；
4. 环形按钮在实际屏幕尺寸下是否需要继续微调。
