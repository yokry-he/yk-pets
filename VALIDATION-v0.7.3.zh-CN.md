## 3. 自动测试

结果：`44/44` 通过。

覆盖范围：

- WebGL 不可用、上下文丢失、低 FPS、长任务、低内存、低电量、Save-Data 和减少动态效果；
- 降级/恢复投票、冷却、状态快照迁移和资源释放；
- 3D 创建失败立即切换 2D，且不会发生切换队列死锁；
- 3D 恢复失败时保留当前健康的 2D 渲染器；
- 无贴图 Canvas 2D 云狐绘制、动画、快照和 Web Component；
- 五个 Local Agent 工具声明、授权覆盖、拒绝优先、一次性确认和次数限制；
- 工具输入/输出校验、强制执行超时和审计记录；
- 插件清单、语义版本、依赖拓扑、循环检测和能力版本；
- 插件通配符权限不得扩大宿主 allowlist；
- 独占能力冲突，以及能力提供者先于消费者激活。

## 6. 当前边界

上一会话使用的临时 v0.7.2 源码目录未延续到当前运行环境，GitHub 连接中也没有可访问仓库。因此本次交付是**可独立验证、可复制合并的新增目录包**，而不是已写入原 monorepo 的提交。

合并时需要将本包的 `packages/*`、相关脚本与文档放入 v0.7.2 仓库，并让现有 Three renderer 实现 `PetRendererFactory` 接口。浏览器真实 WebGL 上下文、低电量设备和扩展宿主中的最终视觉切换仍应在目标项目内进行一次人工验收。
