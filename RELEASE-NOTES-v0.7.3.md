# YK Pets v0.7.3

## 中文

- 新增 WebGL/低性能自动 2D 降级与恢复；
- 新增程序化 Canvas 2D 云狐；
- 新增 Agent 工具能力声明、权限、确认和审计；
- 为 v0.7.2 的五个 Local Agent 工具补齐标准能力声明；
- 新增 `yk-pets.plugin/v1`、语义版本兼容、依赖与能力治理；
- 新增统一平台导出包；
- 浏览器扩展稳定版本保持 `0.6.10`。

## English

- Added adaptive WebGL/low-performance 2D fallback and recovery;
- Added a procedural Canvas 2D cloud fox;
- Added Agent tool declarations, permissions, confirmations, and audit records;
- Added standard declarations for the five v0.7.2 Local Agent tools;
- Added `yk-pets.plugin/v1`, semantic compatibility, dependency and capability governance;
- Added a convenience platform export package;
- Kept the stable browser extension at `0.6.10`.
