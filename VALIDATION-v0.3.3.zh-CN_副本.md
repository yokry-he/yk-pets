# NOVA Browser Agent v0.3.3 验证报告

## 已验证

- 功能、动作、工程共用固定六槽位，从左下角开始填充。
- 当前模式默认只显示一个图标。
- 点击模式图标展开三个模式图标，选择后自动收回。
- 点击宠物、点击菜单外部或按 Esc 均可关闭菜单。
- 菜单关闭时模式入口、环形按钮、分页和 Tooltip 同步隐藏。
- 上一次使用的模式写入扩展本地存储。
- 所有功能、动作和工程按钮保持纯图标，中文说明仅在 Tooltip 中显示。
- 思考气泡与菜单保持独立，并拥有更高层级。
- 动作注册表保留每页六项与分页能力。
- Chrome 包中已移除 v0.3.2 菜单脚本，只加载 v0.3.3 菜单脚本。

## 静态检查

- 双语代码注释检查通过。
- 宠物功能协议检查通过。
- 星云、尾巴、动作、提示、思考气泡检查通过。
- 菜单交互、注册表、极简菜单专项检查通过。
- 12 个 Vue 文件的 TypeScript 脚本语法检查通过。
- CSS 花括号与 Chrome 生产 JavaScript 语法检查通过。
- Manifest 版本和压缩包完整性检查通过。

## 构建边界

容器无法访问 npm registry，`pnpm install` 返回 `EAI_AGAIN`，因此未从空依赖目录重新执行完整 WXT 构建。可加载 Chrome 包以已验证的 v0.3.2 生产构建为基底，移除旧菜单运行脚本后加入单一的 v0.3.3 运行脚本。

容器中的 Chromium 无法初始化 EGL/WebGL，因此未完成像素级 3D 截图验证；布局安全间距通过坐标断言和源码检查验证。
