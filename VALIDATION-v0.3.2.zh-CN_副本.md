# NOVA Browser Agent v0.3.2 验证报告

## 本次范围

- “功能 / 动作 / 工程”三个同级模式。
- 三类菜单按钮全部采用纯图标。
- 中文名称和说明使用独立 Tooltip。
- 工程能力不再展开网页内大型面板，统一进入右侧 Side Panel。
- 功能、动作、工程菜单统一由注册表驱动。
- 每个模式每页最多显示 6 项，并预留页码与滚轮分页。
- 点击宠物统一显示或隐藏切换器、按钮、分页和 Tooltip。
- 重新打开菜单时保留上一次使用的模式。

## 运行时模拟验证

使用 Chromium Headless 构造旧版 Shadow DOM 环境并注入 v0.3.2 运行时代码，实际验证：

- 顶层模式为“功能 / 动作 / 工程”。
- 功能模式显示 2 个图标。
- 动作模式显示 6 个图标。
- 工程模式显示 5 个图标。
- 动作按钮点击会发出 `nova:motion` 事件。
- Tooltip 横向显示，宽度大于 150px，高度小于 100px。
- 关闭菜单后切换器和按钮同步隐藏。
- 不再显示旧版网页内工程大面板。

交付内容包含：

1. 完整的 v0.3.2 Vue / WXT 源码。
2. 基于此前可运行生产包整理的 v0.3.2 Chrome 兼容包。
3. 兼容包已移除 v0.3.0 和 v0.3.1 的叠加运行时脚本，替换为单一、幂等的 v0.3.2 注册表菜单脚本。
