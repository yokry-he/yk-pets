# NOVA Browser Agent v0.6.9 验证报告

## 修复范围

重新生成并替换 v0.6.8 中实际为静音波形的18条中文动作语音，保持现有 Web Audio 播放链路不变。

## 音频验证

- 18/18 个 MP3 可被 FFprobe 和 FFmpeg 解码。
- 文件时长介于 2.50 秒至 5.30 秒。
- 平均音量介于 `-18.3 dB` 至 `-17.6 dB`。
- 峰值介于 `-4.8 dB` 至 `-2.8 dB`。
- 新增的音频回归会在平均音量低于 `-35 dB` 或峰值低于 `-20 dB` 时失败。

## 工程验证

- v0.6.9 审计、宠物视觉与语音专项检查：12/12 通过。
- 双语文档与代码注释检查通过。
- Extension Vue/TypeScript 类型检查通过。
- Chrome MV3 生产构建与 ZIP 生成通过。
- 生产包内18个 MP3 与源码资源逐字节一致。
- ZIP 完整性检查通过。

## 手动检查

请先直接试听交付的 `greeting.mp3`。确认本机可以播放后，重新加载 v0.6.9 扩展并刷新目标网页，再点击招呼、吃饭、云朵午睡和尾光等动作验证浏览器内播放。
